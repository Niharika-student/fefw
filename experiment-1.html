HTML (HyperText Markup Language)
         HTML is the standard markup language used to design and structure web pages.
        It provides the basic framework of a website, which can be enhanced with CSS and JavaScript.
        Features of HTML
           1.Simple and easy to learn.
            2.Uses tags to define elements (e.g., headings, paragraphs, links, images).
           3.Platform independent runs on any browser.
           4.Supports hyperlinks to connect web pages.
           5.Forms the backbone of all websites.
